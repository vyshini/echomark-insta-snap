# Social Media Profile Scraper

A Python-based social media scraping tool that fetches real-time public profile data from **Instagram**, **LinkedIn**, and **Snapchat** using third-party APIs.

### Features

* 📷 Instagram scraping using **Hiker API**
* 💼 LinkedIn scraping using **Apify**
* 👻 Snapchat scraping using **RapidAPI**
* Fetches live public profile information
* Modular and easy-to-maintain Python code
* Error handling and API response validation

### Tech Stack

* Python
* Hiker API
* Apify
* RapidAPI

> **Note:** This project is intended for educational and research purposes. It only retrieves publicly accessible data through the respective APIs and should be used in accordance with each platform's terms of service.
